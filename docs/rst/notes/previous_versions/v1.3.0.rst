Version v1.3.0
==============

This release includes the following **features**:

* Update documentation section corresponding to quitting the application
* Avoid storing characters before finishing execution of echo command
* Update Spy Listeners

This release includes the following **CI improvements**:

* Bump `setuptools` in docs
* Remove deprecated Windows 2019 runner
* Add test reports to CI
* Use `tsan` binaries for `TSAN` workflow
